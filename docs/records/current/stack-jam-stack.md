## [Jam Stack](https://jamstack.org/)

> Requirements

- full html static rendering

- strict typescipt, including typed scss
- markdown
- prefetching
- explicit data & code splitting
- responsive images

> Why

- blazing speed, a11y, s
  eo
- cost: cdn vs server run time
- because 2006 was right
